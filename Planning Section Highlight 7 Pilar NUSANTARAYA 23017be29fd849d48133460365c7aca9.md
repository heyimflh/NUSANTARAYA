- Produk ini punya kedalaman, bukan hanya tampilan visual.

Untuk juri, section ini harus memberi kesan:

- produk punya identitas kuat,
- nama NUSANTARAYA tidak sekadar branding,
- fitur-fitur disusun berdasarkan kerangka konseptual,
- web ini layak disebut “Digital Twin Nusantara”.

### Kenapa layout ini cocok?

- Tidak monoton seperti 7 card sejajar.

- Mudah responsive ke tablet dan mobile.
- Cocok dengan konsep “pintu/portal” seperti [Thorgal](https://www.thorgal.com/).

## 6.2 Layout alternatif: “Horizontal Scroll Portal”

Jika ingin lebih cinematic dan modern:


## 7. Detail Struktur Section

## 7.1 Section wrapper

### Fungsi

Membungkus section dengan suasana transisi dari peta menuju narasi budaya.

### Style

- Background: `#F8F4EA` atau `#FFFDF8`
- Overlay pattern: batik/tenun opacity 4–6%
- Radial glow lembut di tengah
- Padding desktop: `120px 0`
- Padding tablet: `96px 0`
- Padding mobile: `72px 0`
- Max width: `1200px`

### Visual mood

Terang, lapang, kuratorial, premium, seperti halaman koleksi budaya.

---

## 7.2 Section header

### Layout

Header dibuat centered atau split.

### Rekomendasi

Gunakan **centered editorial header** agar section terasa sebagai bab baru.

```
[Tujuh Pilar Eksplorasi]

Satu Peta, Tujuh Cara
Menjelajahi Nusantara

NUSANTARAYA dibangun dari tujuh pilar...
```

### Style

- Eyebrow: uppercase kecil, letter spacing tinggi.
- Heading: serif besar.
- Subheading: Inter, max width 720px.
- Divider ornamen kecil di bawah heading.

### Ukuran

| Elemen | Desktop | Mobile |
| --- | --- | --- |
| Eyebrow | 13–14px | 12px |
| Heading | 56–72px | 36–44px |
| Subheading | 18px | 15–16px |
| Max width text | 760px | 100% |

---

## 7.3 Portal grid

### Grid desktop

Gunakan CSS grid 12 kolom.

```
Row 1:
- Sejarah card: span 5 col, height 460px
- Aksara card: span 3.5 col, height 220px
- Narasi card: span 3.5 col, height 220px

Row 2:
- Sejarah card masih tinggi
- Tradisi card: span 3.5 col
- Alam card: span 3.5 col

Row 3:
- Rasa card: span 3 col
- Yatra card: span 3 col
- Closing connective card: span 6 col
```

### Grid lebih mudah

Jika ingin lebih simpel:

```
Desktop:
- 1 featured card full width row atas
- 6 cards grid 3x2 di bawah

Tablet:
- 2 kolom

Mobile:
- 1 kolom atau horizontal scroll
```

### Rekomendasi teknis

Untuk vibe coding, pilih struktur:

```
Featured card: Sejarah
Grid 6 cards: Aksara, Narasi, Tradisi, Alam, Rasa, Yatra
Closing card: Semua pilar terhubung ke Nusa Map
```

Ini lebih mudah dan tetap premium.

## 10. Interaction Design

## 10.1 Default state

- Featured card aktif: **Sejarah**
- 6 card lain tampil lebih kecil.
- Semua card terlihat lengkap tanpa perlu hover.
- Closing card menampilkan hubungan ke Nusa Map.

---

## 10.2 Hover card

Saat hover:

1. Card naik 4–6px.
2. Icon sedikit rotate atau scale.
3. Watermark huruf pilar bergeser halus.
4. Chips fitur muncul lebih jelas.
5. Border berubah ke warna pilar.
6. CTA arrow bergerak ke kanan.

---

## 10.3 Active pilar

Jika ingin lebih interaktif:

- User klik card.
- Card menjadi active.
- Featured area berubah sesuai card yang dipilih.
- Desktop: card active tampil lebih besar.
- Mobile: active card detail muncul di bawah carousel.

### Catatan

Untuk MVP, tidak wajib ada active state kompleks. Hover saja sudah cukup.

---

## 10.4 Scroll animation

Saat section masuk viewport:

1. Eyebrow fade-up.
2. Heading fade-up.
3. Subheading fade-up.
4. Featured card scale/fade.
5. 6 card kecil muncul stagger.
6. Closing card muncul terakhir.

## 20. Risiko dan Mitigasi

| Risiko | Dampak | Mitigasi |
| --- | --- | --- |
| Section terlihat seperti grid fitur biasa | Kurang premium | Gunakan featured card + editorial grid, bukan 7 card sama rata. |

| Mobile terlalu panjang | Scroll melelahkan | Gunakan compact card dan hide detail panjang di mobile. |
| Animasi berat | Homepage lag | Gunakan Framer Motion ringan dan reduced motion. |
| Ikon tidak konsisten | Design terasa campur aduk | Gunakan satu style icon saja: line/duotone. |
